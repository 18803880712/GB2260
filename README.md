# 县及县以上行政区划代码

![GB/T 2260](http://img.shields.io/badge/GB%2FT-2260-blue.svg?style=flat)

中华人民共和国国家标准 GB/T 2260 行政区划代码

数据来源： <http://www.stats.gov.cn/tjsj/tjbz/xzqhdm/>

你可以利用国家标准数据构建自己项目的数据库。
